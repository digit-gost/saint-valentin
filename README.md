# ❤️ Saint Valentin

A creative and interactive Valentine's Day web project built using HTML, CSS, and JavaScript.

This project was designed as a personalized digital gift, combining animations, dynamic elements, and ambient sound effects to create a romantic and immersive experience.

---

## 💖 Project Concept

The goal of this project was to:

- Create a unique digital surprise
- Practice advanced JavaScript interactions
- Integrate ambient audio effects
- Design a visually emotional user experience

---

## ✨ Features

- Romantic animations
- Dynamic text effects
- Background ambient sound
- Interactive buttons
- Smooth transitions
- Responsive layout

---

## 🔊 Audio Integration

The project includes an ambient sound effect to enhance the emotional atmosphere.  
The sound is controlled using JavaScript audio manipulation.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations & transitions)
- JavaScript (Vanilla JS)
- Audio API
